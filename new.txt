# 🚨 The error

Kubernetes rejected your Kafka resource:

```
unknown field "spec.entityOperator.topicOperator.reconciliationIntervalSeconds"
unknown field "spec.entityOperator.userOperator.reconciliationIntervalSeconds"
```

This means **Strimzi 0.41 no longer supports those fields** in the `entityOperator`.

Those options existed in older versions but were removed.

---

# ✅ Fix

Edit **`kafka-cluster.yaml`**.

Replace this block:

```yaml
entityOperator:
  topicOperator:
    reconciliationIntervalSeconds: 90
  userOperator:
    reconciliationIntervalSeconds: 90
```

with simply:

```yaml
entityOperator:
  topicOperator: {}
  userOperator: {}
```

So the final part becomes:

```yaml
entityOperator:
  topicOperator: {}
  userOperator: {}
```

# 🧠 Small insight (important)

In **Strimzi 0.41 + KRaft mode**:

* `KafkaNodePool` → defines storage + roles
* `Kafka` → defines cluster configuration
* `entityOperator` → just enables operators (no tuning fields)

So the **minimal valid structure** is:

```
KafkaNodePool
KafkaNodePool
Kafka
```

which you already have.
