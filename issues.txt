You switched to:

It expects a **dict config**, not just a URL string.

## ✅ The REAL fix (this will solve it)

Replace your `CHANNEL_LAYERS` with this:

```python
CHANNEL_LAYERS = {
    "default": {
        "BACKEND": "channels_redis.core.RedisChannelLayer",
        "CONFIG": {
            "hosts": [
                {
                    "address": (REDIS_HOST, int(REDIS_PORT)),
                    "password": REDIS_PASSWORD,
                }
            ],
        },
    },
}
```

---

## 💡 Why this works

* `channels_redis` internally uses `redis.asyncio`
* When you pass:

  * `"address"` → host/port
  * `"password"` → it AUTHs correctly
* When you pass URL → sometimes ignored / misparsed depending on version

👉 That’s why your direct `redis.Redis(...)` worked, but Channels didn’t.
